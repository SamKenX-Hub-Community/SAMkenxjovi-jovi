Audio Sampler that only plays 5 words: `JOY`, `DIVI`, `JON`, `BON` and `JOVI`.

Play it here: [https://amagi.dev/jovi-jovi/](https://amagi.dev/jovi-jovi/)


## Development

First, install Rust and Trunk following these instructions.

- https://www.rust-lang.org/tools/install
- https://trunkrs.dev/

Next, clone this repo and launch the dev server using trunk.

```bash
# Clone this repo
git clone https://github.com/fand/jovi-jovi
cd jovi-jovi

# Launch dev server
trunk serve --open
```

Then open [localhost:8080](http://localhost:8080).
